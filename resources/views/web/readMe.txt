views目录下:
	admin: 后台管理页面
	Home: 前台博客页面,头部尾部文件引用layouts目录下的文件
	layouts: 是前台博客头部文件,尾部文件
	web: 是存放个人前端作品页面的目录
	product: 是集中所有个人前端作品展示的一个HTML页面,及各种页面图片及链接
	special: 是存放特效小示例的目录, 里面样式,图片,HTML都有
	jsFile: 是存放自己平时写的js特效的文件, 里面的特效都是独立的文件
	AVFile: 是存放自己平时写的音频/视频特效的文件, 里面的特效都是独立的文件
	JQFile: 是存放jquery特效的文件目录
